# 🚗 Sistema de Concessionária (Projeto de Estudo)

Este projeto foi desenvolvido como uma prática pessoal de aprendizado, com o objetivo de reforçar conhecimentos em desenvolvimento de software e entender melhor como um sistema é planejado e construído do início ao fim.

O sistema simula o funcionamento de uma concessionária, permitindo o gerenciamento de clientes, veículos em estoque e operações relacionadas a test drive.

---

## 📌 Objetivo do Projeto

O principal objetivo deste projeto não é acadêmico, mas sim educacional e prático. Ele foi criado para explorar conceitos de programação orientada a objetos, modelagem de sistemas e lógica de desenvolvimento.

---

## ⚙️ Funcionalidades

O sistema conta com as seguintes funcionalidades:

- Cadastro de clientes  
- Atualização de dados de clientes  
- Consulta de clientes  
- Gerenciamento de estoque de veículos  
- Cadastro de veículos  
- Atualização de estoque  
- Simulação de test drive  
- Consulta de test drive  

---

## 🧠 Modelagem do Sistema

Antes da implementação, o sistema foi planejado utilizando diagramas para melhor organização das ideias:

- Diagramas de Casos de Uso  
- Modelagem Conceitual (UML)  

Esses diagramas ajudaram a definir as funcionalidades do sistema e a forma como o usuário interage com ele, facilitando a tradução do modelo para o código.

---

## 🛠️ Tecnologias Utilizadas

- C# (.NET)
- Programação Orientada a Objetos (POO)
- Lógica de Programação
- UML (Modelagem de Sistemas)
- Testes manuais durante o desenvolvimento

---

## 🧩 Estrutura do Projeto

O projeto foi desenvolvido de forma incremental, com foco em aprendizado. A estrutura segue princípios de organização por entidades e responsabilidades, simulando um sistema real.
